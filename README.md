Projeto criado com [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Iniciando

Primeiro, instale as dependencias do projeto.

```bash
npm install
# or
yarn
# or
pnpm install
# or
bun install
```

Após isso, execute o server de desenvolvimento:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

O resultado vai estar em [http://localhost:3000](http://localhost:3000).
